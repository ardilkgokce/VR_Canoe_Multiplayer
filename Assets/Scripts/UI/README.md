# UI Scripts

Kullanici arayuzu scriptleri.

## Scriptler

### MainMenuUI.cs
- Ana menu
- Photon baglanti durumu
- Oda listesi

### LobbyUI.cs
- Oyuncu listesi
- Hazir durumu
- Oyun baslatma butonu

### NameInputUI.cs
- Oyuncu isim girisi
- Isim kaydetme (PlayerPrefs)
- Karakter limiti

### ScoreboardUI.cs
- Canli skor tablosu
- Takim skorlari
- Siralama gosterimi

### TimerUI.cs
- Geri sayim gosterimi
- Kalan sure
- Animasyonlar

### GameOverUI.cs
- Sonuc ekrani
- Kazanan gosterimi
- Tekrar oyna butonu

### SpectatorUI.cs
- Izleyici kontrolleri
- Kamera degistirme
- Takim secimi

## VR UI Notlari
