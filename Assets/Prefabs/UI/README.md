# UI Prefabs

Kullanici arayuzu prefablari.

## Prefablar

### MainMenuCanvas.prefab
- Ana menu paneli
- Baglanti durumu
- Butonlar

### LobbyCanvas.prefab
- Oyuncu listesi
- Hazir butonlari
- Oda bilgileri

### InGameCanvas.prefab
- Skor tablosu
- Timer
- Mini harita

### GameOverCanvas.prefab
- Sonuc ekrani
- Siralama
- Tekrar oyna

### NameInputPanel.prefab
- Isim girisi
- Klavye
- Onay butonu

### ScorePopup.prefab
- Coin toplama popup
- +10 animasyonu
- Fade out

### CountdownPanel.prefab
- 3-2-1-GO!
- Buyuk rakamlar
- Animasyon

## VR UI Kurallari

```
- Canvas: World Space
- Scale: 0.001 (1 unit = 1mm)
- Distance: 1-2 metre
- Event Camera: XR Camera
```

## Notlar

- TextMeshPro kullan
- Raycaster ayarla VR icin
- Canvaslar oyuncuya baksIn
